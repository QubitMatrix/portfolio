## Overview

Co-Lancer is a freelancer collaboration platform that streamlines project discovery, payment, and feedback for freelancers. The platform simplifies the process for freelancers to find projects aligned with their skills and ensures secure compensation. It also enables clients to provide feedback, fostering motivation and ensuring top-quality service. The platform's primary goal is to provide freelancers with an accessible portal for diverse projects and make client-freelancer collaboration more straightforward.

## Key Features

- Register as freelancer or client
- Login Page
- Explore and Join projects
- Handoff and Review projects
- Create new projects
- Payment
- Feedback
- Chat (in-progress)
- Monthly Recap

## Technologies Used
- MERN Stack
- Tailwind CSS
- MySQL
- AWS RDS

> The project was developed with mitigations for majority of web application attacks that can be employed on web applications, database and web servers. Namely SQLi, Directory Traversal, Unauthorized entry, Brute Forcing, Denial of Service, etc.
